Kiosk Project

C# Program


Check it out on Github!

Project Description

This project simulates a transaction kiosk. It accepts cash and credit from the user, uses the Luhn algorithm for card number verification, and logs transactions in a .log file every time a successful transaction is made.

Project Role / Highlights

The Kiosk program was a project assigned to me while attending Mississippi Coding Academies to evaluate our comprehension of C# as well as an opportunity to showcase mine and other MCA student's coding ability to employers from the South Mississippi region. Below is an evaluation conducted by Mississippi Coding Academies that graded the "Code Readability", "Developer Documentation", "Program Performance", and "Presentation Quality".
